# QUIZ Game

## Setup Instructions

1. Clone the repository or download the source code.
2. Create a virtual environment:
   ```bash
   python -m venv .venv
3. Activate your virtual environment
  (on mac)
  ```bash
  source .venv/bin/activate
4. Install dependencies
  ```bash
  pip install -r requirements.txt
5. Run program
  ```bash
  python main.py
